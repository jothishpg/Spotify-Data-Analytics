# 🎧 Spotify Data Analytics & Database Management

This project focuses on analyzing Spotify track data using SQL and Python by designing a relational database and extracting meaningful insights through optimized queries.

---

## 🚀 Features
- Designed a normalized MySQL database schema for storing Spotify track data  
- Performed data ingestion from CSV files using Python  
- Executed advanced SQL queries (joins, aggregations, filtering, CASE statements)  
- Analyzed trends such as:
  - Most popular tracks  
  - Average popularity  
  - Duration-based filtering  
- Demonstrated query optimization and efficient data retrieval  

---

## 🛠️ Tech Stack
- **Database:** MySQL  
- **Languages:** SQL, Python  
- **Libraries:** Pandas, NumPy  
- **Tools:** Git, GitHub  

---

## 📊 Key Concepts Used
- Relational Database Design  
- Normalization  
- SQL Aggregations (`GROUP BY`, `AVG`, etc.)  
- Data Analysis & Processing  
- Query Optimization  

---

## 🎯 Objective
To build a data-driven system that efficiently stores, processes, and analyzes Spotify data to derive actionable insights.
